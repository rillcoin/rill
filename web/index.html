The Decay Mechanism

Sigmoid Decay, Not Confiscation

Decay is gradual, predictable, and purely mathematical. No governance decisions, no arbitrary thresholds ‚Äî just integer arithmetic with 10‚Å∏ precision applied consistently to every UTXO.

Integer-Only Consensus Math

All calculations use fixed-point u64 arithmetic. No floating point ‚Äî every node on the network computes identical results deterministically.

üìê

Exponential Cluster Indexing

UTXOs are mapped to logarithmic clusters. Only the excess above each cluster boundary decays ‚Äî the base amount is always preserved.

Self-Balancing Supply

As large holders decay, the decay pool grows, rewarding miners more. This creates a natural equilibrium that keeps the supply in motion.
